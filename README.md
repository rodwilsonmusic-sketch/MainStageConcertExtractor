# MainStage Concert Extractor Prototype

This is a standalone Python utility designed in parallel with the Midi Assistant project. 
Because this is entirely separate from the Swift UI codebase, you can assign an AI Agent here to aggressively rip through XML and Plist files dynamically without any risk of breaking the main Swift application!

## How to use Context
When opening a new IDE window or chat thread for this project, instruct the agent:
> "We are building a standalone Python prototype to extract assignment mappings from a MainStage .concert file. Please modify `extractor.py` to parse its XML."
